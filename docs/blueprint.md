# **App Name**: SkillSphere

## Core Features:

- Mock User Authentication: Implement a login page with email and password fields. Assume successful login redirects to the home page.
- User Profile Listing: Display a list of user profiles with their offered and wanted skills, each card should have a request button.
- Swap Request Modal: Implement a popup modal to display 'Swap Request' details including accept and reject buttons
- User Profile Page: Allow users to view their profile showing skills offered/wanted, availability, and profile visibility. Provide an 'Edit Profile' button.
- Edit Profile Form: Create a form to edit user profile details, including skills, availability, and profile visibility.
- Request Swap Page: Develop a 'Request Swap' page with dropdowns to select offered and wanted skills and a message input area.
- Search and Pagination: Include a search bar component for quick filtering, plus pagination to ensure a good user experience with long lists.

## Style Guidelines:

- Primary color: Darker blue (#898AC4) for text and primary elements.
- Secondary color: Medium blue (#A2AADB) for primary accents and borders.
- Accent color: Light blue (#C0C9EE) for secondary accents.
- Background color: Lightest beige (#FFF2E0) for the background and light accents.
- Font: 'Inter' (sans-serif) for all text elements to ensure a modern and readable interface.
- Use simple, rounded icons that match the color scheme to represent skills and other actions.
- Employ rounded corners on all elements to maintain visual consistency.
- Incorporate subtle transitions and animations (using Tailwind's animation utilities) on interactive elements like buttons and modal transitions for better engagement.
